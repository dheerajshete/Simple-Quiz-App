# Simple Quiz App

A simple quiz application built using HTML, CSS, and JavaScript.

## Features

- Multiple choice question format
- Radio button answer selection
- Answer validation
- Correct/Wrong feedback
- Clean and beginner-friendly code structure

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

```
Simple-Quiz-App
│
├── quiz.html
├── quiz.css
└── quiz.js
```
